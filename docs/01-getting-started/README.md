## Overview

Mail History is a Laravel package that automatically tracks all emails sent through your application, whether sent via Mailable classes or Notification classes. It provides:

- **Automatic Email Tracking** - Captures email metadata without changing your existing code
- **Status Monitoring** - Tracks email sending and sent states
- **Hash-based Identification** - Unique identifiers for each email
- **Queue Support** - Works seamlessly with Laravel's queue system

## Prerequisites

Before installing Mail History, ensure you have:

- PHP 8.1 or higher
- Laravel 9.x, 10.x, or 11.x
- Composer installed
